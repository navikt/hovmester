03 / Arbeidsobjekter issues + templates + PR 03 / 14

Issues er arbeidsordre for agenter

Hovmester hjelper ikke bare med kode. Den hjelper teamet å lage bedre oppgaver.

EPIC implementer ny visning ├─ TASK backend-kontrakt ├─ TASK frontend fra Figma └─ TASK test + review hver task = fersk kontekst hver PR = tydelig historie

07 / Når trenger du orkestrator? context overflow + review 07 / 14

Orkestrering gir verdi når én agent blir flaskehalsen

Flere roller trengs

Plan, backend, frontend og review er ulike jobber. Samme chat er ofte feil arbeidsflate.

Du vil ha kryssmodell-review

Modeller har ulike blindsoner. En annen modell kan lese koden med andre briller.
